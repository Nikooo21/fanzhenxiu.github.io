第2章 变量与常用数据类型

变量是存储数据的容器，Python常用数据类型分为：字符串、整数、浮点数、布尔类型。本章学习变量定义、赋值与类型查看。

学习要点：

1. 变量命名规则与赋值方式

2. 字符串、整数、浮点数、布尔类型使用

3. 使用 type() 查看数据类型

示例代码

# 定义变量
name = "樊珍秀"
age = 20
height = 1.65
is_student = True

# 输出变量
print(name)
print(age)
print(height)
print(is_student)

# 查看数据类型
print(type(name))
print(type(age))
