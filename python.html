Python 是一门简单易学、功能强大的编程语言，拥有高效的高级数据结构和简单有效的面向对象编程方法。Python 的语法和动态类型，以及解释型语言的本质，使它成为多数平台上写脚本和快速开发应用的理想语言。

Python 核心要点

  • • 解释型语言：无需编译，直接运行代码
  • • 动态类型：无需声明变量类型，运行时自动判断
  • • 缩进语法：使用缩进表示代码块，强制代码规范
  • • 丰富的库：NumPy、Pandas、Matplotlib 等数据分析库
  • • 跨平台：可在 Windows、Mac、Linux 上运行

示例代码

# 第一个Python程序
print("Hello, 樊珍秀!")

# 变量定义
name = "樊珍秀"
major = "商务数据分析与应用"
print(f"我是{name}，专业是{major}")

# 简单计算
a = 10
b = 20
print(f"a + b = {a + b}")
